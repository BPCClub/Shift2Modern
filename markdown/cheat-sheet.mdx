# 速查表

此速查表提供了一些常用的 *Markdown* 语法的快速参考。受限于篇幅，本速查表不能完整地列出 *Markdown* 语法的所有特性，如果您想了解更多 *Markdown* 语法的特性，请参阅 [Markdown 基础语法](./basic-grammar.md)和 [Markdown 额外语法](./extra-grammar.md) 。

|   元素   | 语法                                                                          |
| :------: | :---------------------------------------------------------------------------- |
|   标题   | `# 一级标题` &lt;br /&gt; `## 二级标题` &lt;br /&gt; `### 三级标题`                             |
|   粗体   | `**粗体**`                                                                    |
|   斜体   | `*斜体*`                                                                      |
|   引用   | `> 引用`                                                                      |
|   代码   | `` `代码` ``                                                                  |
| 有序列表 | `1. 有序列表` &lt;br /&gt; `2. 有序列表` &lt;br /&gt; `3. 有序列表`                             |
| 无序列表 | `- 无序列表` &lt;br /&gt; `- 无序列表` &lt;br /&gt; `- 无序列表`                                |
|   链接   | `[链接文本](链接地址)`                                                        |
|   图片   | `![图片描述](图片地址)`                                                       |
|  代码块  | ```#include&lt;stdio.h&gt;``` &lt;br /&gt;```int main()``` &lt;br /&gt; ```{printf("HelloWorld");}``` |
|   脚注   | `脚注[^1]` &lt;br /&gt;`[^1]: 脚注内容`                                                |
|  删除线  | `~~删除线~~`                                                                  |
| 任务列表 | `- [x] 已完成` &lt;br /&gt; `- [ ] 未完成`                                             |
| 定义列表 | `术语 1` &lt;br /&gt; `: 这是该术语的定义内容。`                                       |
